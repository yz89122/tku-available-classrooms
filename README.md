# TKU Available Classrooms

## Run

1. Install dependencies

   ```bash
   npm install
   ```

2. Run script

   ```bash
   npm start
   ```

3. The results file are located in `./output`, the name of the file is the name of the building in TKU. Ex. `E.csv` is the New Engineering Building.
